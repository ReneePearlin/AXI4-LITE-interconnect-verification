# AXI4-Lite Interconnect Design and Verification

## Overview

This project focuses on the design and verification of an AXI4-Lite Interconnect using SystemVerilog.

The project is being developed from first principles with an emphasis on learning digital design, SoC communication architectures, bus protocols, RTL development, verification methodologies, and engineering documentation.

---

## Project Objectives

- Understand SoC communication architectures
- Learn the AMBA protocol family
- Study AXI4-Lite protocol operation
- Learn SystemVerilog for RTL design and verification
- Design an AXI4-Lite Interconnect
- Verify functionality through simulation
- Document the engineering workflow on GitHub
